Conenido del curso

Semana Tema Contenido
Semana 1 HTML
  • HTML significa lenguaje de marcado de hipertexto
  • HTML es el lenguaje de marcado estándar para crear páginas web.
  • HTML describe la estructura de una página Web
  • HTML consta de una serie de elementos
  • Los elementos HTML le dicen al navegador cómo mostrar el contenido
  • Los elementos HTML etiquetan piezas de contenido como "este es un encabezado", "este es un párrafo", "este es un enlace", etc.
Semana 2 CSS
  • CSS significa hojas de estilo en cascada
  • CSS describe cómo se deben mostrar los elementos HTML en la pantalla, el papel o en otros medios
  • CSS ahorra mucho trabajo. Puede controlar el diseño de varias páginas web a la vez
  • Las hojas de estilo externas se almacenan en archivos CSS
Semana 3 JS
  • Java es un lenguaje orientado a objetos y basado en objetos
  • El comportamiento que requiere que el usuario active la página
  • JavaScript no tiene acceso al sistema operativo, por lo que no puede operar recursos del sistema como archivos y registro, y no puede usarse para crear virus y caballos de Troya.
  • El lenguaje de secuencia de comandos también se denomina lenguaje extendido o lenguaje dinámico, que es un lenguaje de programación que se utiliza para controlar aplicaciones de software.
Semana 4 React
  • Composición de componentes.
  • Desarrollo Declarativo Vs Imperativo.
  • Flujo de datos unidireccional.
  • Performance gracias al DOM Virtual.
  • Isomorfismo.
